How To Add or Edit Wiki Pages On Github

This article will guide you on how to create and deploy wiki pages directly on github. Keep in mind that this can also be done via the command line.

Adding A Wiki Page

  1. Login into your github account, and navigate to the main page of the repository.

  2. Under the repository name, click on Wiki

  3. In the upper-right-hand corner, click on New Page

  4. For this tutorial we are using markdown as our format. But this can be changed to another format if you use the Edit mode drop-down menu. And select a different format

  5. Now you can use the editor to add to the page

  6. In order to commit, add a message describing what the file is

  7. To commit the changes, click on Save Page
